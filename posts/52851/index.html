JavaScript里的数据类型

2018-12-18 前端 JavaScript 0 评论

1.简介

JavaScript 有六种数据类型。（ES6又新增了第七种Symbol类型的值）

  • number：整数和小数。
  • string：文本
  • boolean：表示真伪的两个特殊值 true和false
  • undefined：未定义或不存在
  • null：表示空值
  • object：各种值的组合

1.1整数和浮点数

JavaScript内部所有的数字都是64位浮点数形式存储，即使整数也是如此。

容易造成混淆的是，某些运算只有整数才能完成。此时js会自动把64位浮点数转换成32位整数，然后再进行运算。

JavaScript对整数提供四种进制的表示方法：十进制、十六进制、八进制、二进制。

  • 十进制：没有前导0的数值
  • 八进制：有前缀0o或者00的数值。或者有前导0，且只用到0-7八个数字的值。
  • 十六机制：有前缀0x或者0X的值。
  • 二进制：有前缀0b或者0B的值。

有前缀0通常会被视为八进制，但是如果0后面有数字8和9，则会被视为十进制。

1.2字符串

字符串默认只能写在一行，分成多行会报错。
如果必须分行，在每一行的结尾使用反斜杠。

注意：反斜杠后面必须是换行符，而不能有其他字符。
或者可以使用+号链接多个单行字符串。

转义字符串

  • \0 null
  • \b 后退
  • \n 换行
  • \r 回车
  • \t 制表符
  • \‘ 单引号
  • \“ 双引号
  • \\ 反斜杠

JavaScript使用Unicode字符集，JavaScript引擎内部，所有字符都用Unicode表示。

JavaScript提供两个和Base64有关的方法

  • btoa() 任意值转为base64编码
  • atob() base64编码转为原来值

注意：要将非ASCII码字符集转为base64 编码，中间必须插入一个转码环节。

1.3 对象

对象是一组键值对集合，是一种无序的复合数据集合。
读取对象的属性，有两种方法：一种是使用点运算符，一种是使用方括号运算符。
注意：使用方括号运算符，键名必须放在引号里面，否则会被当做变量处理。

查看一个对象的所有属性，可以通过Object.keys方法。
delete命令用于删除对象的属性。
属性是否存在：in运算符
属性的遍历：for … in 循环
